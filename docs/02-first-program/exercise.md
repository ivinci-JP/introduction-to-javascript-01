# 練習問題

1. このプログラムのコードを読み、paiza.ioで実行せずに出力結果を推測してみましょう。推測したら実際に実行して結果を確認してください。
```js
function main() {
    console.log("1");
    console.log("2");
    console.log("3");
}
```

```

learning
JavaScript!
```

3. このプログラムをコピーしてpaiza.ioに貼り付けます。
```js
function main() {
    console.log("Tuesday");
    console.log("Thursday");
    console.log("Wednesday");
    console.log("Friday");
    console.log("Monday");
}
```
以下の出力を行うようにプログラムを修正してください。
```
Monday
Tuesday
Wednesday
Thursday
Friday
```

4. プログラムを実行しようとすると、エラー メッセージが表示されます。
目的の出力が生成されるように、プログラムのエラーを修正してください。
```js
function main() {
    console.log("Tomorrow is rainy");
```
目的の出力:
```
Tomorrow is rainy
```

5. 目的の出力が生成されるように、プログラムのエラーを修正してください。
```js
function main() {
    console.logs("There is a chance of snow");
}
```

6. 目的の出力が生成されるように、プログラムのエラーを修正してください。
```js

    console.log("Cloudy") console.log("Partly Cloudy") console.log("Windy")
}
```
目的の出力:
```
Cloudy
Partly Cloudy
Windy
```

```js
)
```

```
How's the weather today?
```
