# JavaScriptハンズオン 基本編

## このドキュメントについて

本ドキュメントは本格的なプログラミングを始める前の練習として
簡単なJavaScriptの実際に書いて動かしてみる為の手引きです。
限られた勉強会の時間内でJavaScriptのコーディングを学習することを考慮し、
比較的コンパクトな内容にとどめています。

JavaScriptに関する詳細な解説や、最新の情報については[MDN](https://developer.mozilla.org/ja/docs/Web)
を参照するか、適宜検索を掛けるなどして各自で補完して下さい。
JavaScriptに関するチュートリアルは通常、HTMLやCSSなどの技術とセットで扱われることが殆どですが、
本ドキュメントでは出来る限りHTMLやCSSに依存しない、純粋なJavaScriptに関して記述します。

## 対象読者

- プログラミング経験が無い方
- プログラミングを始めるきっかけが欲しい方
- JavaScriptに興味がある方

内容が難しくなり過ぎない様に配慮していますが、そもそも初めてのプログラミングとはある程度は難しいものです。他のプログラミング言語の経験があれば理解の助けになります。
また、プログラミングは習うだけでは身に付きません。自分自身が考えたり調べたりする中で、少しずつ身に付くものです。やる気と根気をもって読み進めて下さい。
